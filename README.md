# AdmissionLens
AdmissionLens analyzes 10 years of university data to improve admissions. It identifies trends in applicant origins, popular majors, and common rejection reasons. By uncovering bottlenecks—like document delays—it helps universities simplify processes and boost efficiency, turning raw data into clear, actionable insights.


## How to Run
1. Install the required libraries: 
   `pip install pandas matplotlib scikit-learn`
2. Run the analysis script:
   `python model.py`
3. The script will generate 'admissions_data.csv' and 'visualization.png'.
