# AdmissionLens
AdmissionLens analyzes 10 years of university data to improve admissions. It identifies trends in applicant origins, popular majors, and common rejection reasons. By uncovering bottlenecks—like document delays—it helps universities simplify processes and boost efficiency, turning raw data into clear, actionable insights.


## Usage Instructions
1. Prepare your data in a CSV file with two columns: 'Months' and 'Applications'.
2. Place your file (e.g., 'university_data.csv') in the same directory.
3. Run the script using your own data:
   `python model.py university_data.csv`
4. The system will automatically generate 'university_analysis.png' based on your input.
